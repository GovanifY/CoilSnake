## Installation

On Windows, just [run the installer](#). On Linux, install from source:

    $ sudo apt-get install python-pip python-dev libyaml-dev python-tk \
                           g++ libboost-filesystem-dev
    $ git clone https://github.com/kiij/CoilSnake.git
    $ cd CoilSnake
    $ make
    $ sudo make install
